SUMO 說明文件 (SUMO User Documentation)

請先看這裡！

本份文件是 SUMO Documentation 於 2022/04/01 版本的快取，旨在將 SUMO Documentation 翻譯成正體中文。如果需要最新的說明文件，歡迎造訪 SUMO Documentation 的官方網站。

"Simulation of Urban MObility (SUMO)" 是一個開放原始碼、微觀車流與持續性的交通模擬套件，旨在支援大型的路網模擬。本軟體可對包括行人或多重情境下進行綜合模擬。

本軟體主要是由德國航空太空中心 (Deutsches Zentrum für Luft- und Raumfahrt e.V.) 內的運輸系統研究所開發。SUMO 的授權條款為 EPL 2.0。The source code may also be made available under the following Secondary Licenses when the conditions for such availability set forth in the EPL 2.0 are satisfied: GPL2 or later.

歡迎使用聯絡我們頁面來分享你使用 SUMO 的結果及詢問問題；針對某些常見問題的答案，或許你可以在常見問題頁面中得到一些幫助。

如果你使用 SUMO 進行車流模擬，你可以在出版品這一頁中和我們分享。

關於引用

如果你需要引用 SUMO，請參考我們目前的出版格式："Microscopic Traffic Simulation using SUMO"; Pablo Alvarez Lopez, Michael Behrisch, Laura Bieker-Walz, Jakob Erdmann, Yun-Pang Flötteröd, Robert Hilbrich, Leonhard Lücken, Johannes Rummel, Peter Wagner, and Evamarie Wießner. IEEE Intelligent Transportation Systems Conference (ITSC), 2018.

你可以自由修改 SUMO 使用者文件；要修改特定頁面，請點選位在網頁右上角的「在 GitHub 上編輯中文版」按鈕，並在修改完後提出 pull 要求。這裡有份關於編輯文章的教學頁面。如有需要，你也可以在自己的電腦上建置 (Build) 這份文件，也可以下載一份副本到你的電腦中。如果你要修改原文版本，請到 SUMO Documentation 了解更多資訊。

SUMO Documentation 會隨著 SUMO 的版本迭代而隨時更新。特定 SUMO 版本的說明文件會隨著你下載的 SUMO 版本一起提供。請到 <SUMO_HOME>/docs/userdoc/index.html 查看該對應該 SUMO 版本的說明文件。

其他工具#

除了 SUMO 套件的主要程式（包括 sumo, sumo-gui, netedit, netconvert 等）外，SUMO 套件尚有 150 多個小應用程式。他們多半與交通路網分析、需求生成、修改需求以輸出分析有關，多數的小程式以 python 撰寫而成。這些小程式都可以在 <SUMO_HOME>/tools 資料夾裡找到。

要查看這些小程式的清單，請參見：

  • 工具索引

以下的連結是一些好用（或重要）的小程式：

  • osmWebWizard：可以使用瀏覽器在極短的時間內，以真實世界的地圖生成路網配置。
  • Interfacing TraCI from Python：用 Python 存取正在進行的 SUMO 模擬。
  • sumolib：用來與 SUMO 路網和 sumo XML 檔案一同運作的 Python 模組。
  • Xml Tools：將 SUMO 模擬結果與 CSV／試算表相互交換的小工具。
  • traceExporter.py：輸出移動軌跡到不同的「軌跡檔案」格式。
  • netdiff.py：比較兩個路網檔案的不同處。
  • Visualization Tools：將一部分的模擬輸出結果以圖形及友善的形式視覺化。
